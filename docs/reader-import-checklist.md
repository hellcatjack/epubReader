# Reader Import Checklist

Use this quick pass when validating EPUB import behavior:

- Select a representative sample EPUB that is valid, readable, and safe to share with maintainers.
- Import the EPUB through the app's library import flow and wait for it to appear in the library.
- Open the imported book from the library entry, not from a direct reader URL.
- Confirm the reader reaches the first readable section and renders its text content.
- Note the EPUB name, import result, and any reader rendering issue in the related issue or PR.
